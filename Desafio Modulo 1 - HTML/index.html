História do HTML

O HTML (HyperText Markup Language), ou "Linguagem de Marcação de Hipertexto", foi criado em 1991 por Tim Berners-Lee, um cientista britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear). A ideia surgiu quando ele propôs a criação de uma rede mundial de documentos interligados para facilitar a troca de informações entre cientistas e pesquisadores de diferentes partes do mundo. Esse projeto seria a base para o que hoje conhecemos como a World Wide Web (WWW).

A Primeira Versão do HTML

Tim Berners-Lee desenvolveu o HTML com um objetivo simples: criar uma linguagem que pudesse ser usada para estruturar documentos e fazer referências a outros documentos, conhecidos como links. A primeira versão do HTML foi muito simples e continha apenas 18 tags. Essas tags permitiam organizar texto, adicionar links e criar listas. O HTML inicial não possuía nenhum formato avançado e estava longe de ser o que conhecemos hoje.

Evolução do HTML

À medida que a Web crescia, o HTML também evoluía. Nos anos seguintes, mais recursos foram sendo incorporados à linguagem para suportar imagens, tabelas e até formulários, tornando-a mais útil para a criação de páginas dinâmicas e interativas.

  • HTML 2.0 (1995): A primeira versão oficialmente padronizada, que adicionou funcionalidades importantes, como tabelas e formulários.

  • HTML 3.2 (1997): Introduziu suporte a elementos de formatação avançada como as tabelas e formulários.

  • HTML 4.01 (1999): Estabeleceu a base sólida para o HTML moderno e a separação de conteúdo e apresentação, permitindo maior controle sobre o layout e a estruturação das páginas.

O HTML5 e a Web Moderna

Em 2008, o HTML5 foi proposto pelo W3C (World Wide Web Consortium) para melhorar a linguagem, acrescentando recursos para suportar multimídia (como áudio e vídeo), gráficos e interatividade nativa sem a necessidade de plugins externos. Com a popularização dos dispositivos móveis e das aplicações web modernas, o HTML5 se tornou a versão definitiva para criar páginas web interativas e responsivas.

O HTML5 também foi projetado para ser mais semântico e acessível, com novas tags como <header>, <footer>, <article>, e <section>, que ajudam a melhorar a estrutura e o entendimento do conteúdo pelas máquinas e pelos usuários.

O que são tags no HTML?

As tags no HTML são elementos fundamentais usados para estruturar e organizar o conteúdo de uma página web. Elas funcionam como comandos que indicam ao navegador o que fazer com determinado conteúdo.

Cada tag é envolvida por sinais de menor e maior (< e >). A maioria das tags possui uma tag de abertura (como <p>) e uma tag de fechamento (como </p>). O conteúdo vai entre essas duas tags.

Como funcionam algumas tags básicas no HTM

  1. <HTML>: A tag <html> é a base de toda página HTML. Ela indica ao navegador que aquele arquivo é um documento HTML. Todo o conteúdo da página deve estar dentro dela.

    Exemplo <html>

  2. <H1> a <H6>: Essas tags são usadas para criar títulos e subtítulos, variando de <h1> (o maior) até <h6> (o menor).

    Exemplo <h1> a <h6>

  3. <P>: A tag <p> é usada para criar parágrafos de texto.

    Exemplo <p>

  4. <A>: A tag <a> é usada para criar links clicáveis que levam a outras páginas ou documentos.

    Exemplo <a>

  5. <IMG>: A tag <img> é usada para exibir imagens. Ela precisa do atributo src (fonte) para indicar o arquivo da imagem e de um atributo alt para descrever a imagem.

    Exemplo <img>

  6. <UL> e <OL>: Essas tags criam listas. A <ul> cria uma lista não ordenada (com marcadores) e a <ol> cria uma lista ordenada (com números).

    Exemplo <ul>
    Exemplo <ol>
